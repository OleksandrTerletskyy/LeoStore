Проект, над яким я на даний момент працюю.
Це має бути простий інтернет магазин з мінімальним функціоналом:
	1. Повинна бути головна сторінка, на якій відображається сітка товарів.
	2. В кожного товару є своя індивідуальна сторінка.
	3. Має бути корзина, в яку можна добавляти товари, видаляти і тд.
		3.1 До корзини має бути швидкий доступ з кожної сторінки магазину.
		3.2 Також повинна бути сторінка, де відображаються всі товари з корзини.
	4. Повинна бути сторінка оформлення замовлення, де користувач має вказати свої дані, вибрати метод оплати і тд.
	5. Повинен бути інтерфейс оперуванням контенту на сайті.
	
На даний момент частково зроблена головна сторінка:
	1. Відображається сітка товарів
	2. При наведенні курсора на товар, з'являється невелике меню, де можна подивитись ціну товару, добавити одиницю товару в корзину (чи видалити).
	3. В хедері є швидкий доступ до корзини.
	4. Реалізовано набір фільтрів : пошук, мінімальна ціна, максимальна ціна, сортування по зростанню чи по спаданню ціни, фільтрування за категоріями (тегами).

Також частково є зробленою індивідуальна сторінка товару:
	1. Відображення інформації про товар, можливість добавити товар в кошик.
	2. Галерея, де можна переглянути фотографії товару.


Для запуску потрібно змінити connection string в web.config.
